Background

This dashboard explores economic inactivity in the UK over the last three decades, utilising data from the Labour Force Survey. It unveils inactivity rates by reason (e.g., retirement, ill health) and analyses gender disparities. The latest quarter's data provides a current snapshot of this trend.

Dataset issues

There are some considerations that should be noted about the dataset.

  • change in weighting from September 2022 resulting in step change in counts.
  • there sample size of discouraged workers is small, therefore an estimate is used.

A change of contrast highlights the start of such data problems in all time-series plots
